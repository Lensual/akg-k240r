# akg-k240r
仿制k240r mini XLR接口处的外壳零件

![akg-k240r v1.1](https://github.com/Lensual/akg-k240r/raw/master/img/akg-k240r%20v1.1.png)

v1.0与v1.1区别仅在于那个圆柱套套筒为了3d打印壁厚增加了

但是v1.1版本依然存在缺陷：
* 攻丝孔小了，热熔胶解决
* 上方导线预留孔间距错误，美工刀解决
* 没有仿出来原版的凹槽

下面图片都是v1.1版本的，3d打印 尼龙材质
